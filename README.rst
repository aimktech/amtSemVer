amtSemVer
=========

|pypi| |coverage| |license|

----

*amtSemVer* is a Python 3 package that provides a very simplistic support
for the `Semantic Versioning`_ 2.0.

----

Basic Usage
-----------

Install with pip:

.. code:: bash

    pip install amtsemver
